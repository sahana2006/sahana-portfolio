# Sahana Portfolio

A modern developer portfolio built with React, Vite, Bootstrap, Framer Motion, and React Icons.

## Run locally

```bash
npm install
npm run dev
```

## Notes

- Replace placeholder social links in `src/data/portfolioData.js` and component files.
- Replace `public/resume.txt` with your real resume file if you want a polished download button.
